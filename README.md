## ▶️ How to Run

1. Go to [Releases](github.com/httpsnaziha/Adhan-time-desktop-app/releases)
2. Download `Adhan.exe`
3. Double-click to run
4. If Windows shows a security warning, click **"More info"** → **"Run anyway"**
